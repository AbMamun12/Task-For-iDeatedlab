A sleek and modern Flutter application to help you organize your tasks efficiently, track daily activities, and manage your schedule seamlessly. Designed with a clean UI, light & dark theme support, and interactive task management features.

## Features

- **Splash Screen Animation**: Smooth fade & scale animation for an engaging start.
- **Home Dashboard**: Displays current day, week number, and a summary header card.
- **Weekly Calendar**: Horizontal scrollable week view to quickly navigate days.
- **Task List Management**:
  - View all tasks in a clean card layout
  - Add, update, delete, and toggle task status
- **Bottom Navigation Bar**:
  - Interactive nav items with icon highlight
- **Theme Support**:
  - Switch between light and dark mode dynamically
- **Responsive UI**: Works perfectly on different screen sizes
